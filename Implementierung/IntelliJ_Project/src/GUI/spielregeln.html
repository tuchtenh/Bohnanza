Spielsteueung:

  • Die Steuerung des Spiels erfolgt über Buttons, z.B. "Handeln", "Nächste Phase" oder auch "Bestenliste"
  • Mit den Hand- und Bohnenfeldkarten wird mittels Mausklick interagiert
  • Klickt man z.B. in der eigenen Phase 1 auf die Handkarten, so öffnet sich ein Fenster zum Anbauen
  • Gleiches geschieht mit den Ziehkarten und Handelkarten in Phase 3
  • Mit einem Klick auf das Bohnenfeld, wird dieses "geerntet"
  • Die Zuweisung neuer Karten oder gehandelter Karten erfolgt automatisch - ein manuelles Ziehen ist nicht möglich

------------------------------------------------------------------------

Spielablauf:

  • Der Startspieler (mit Startspielkarte) beginnt. Die Startspielkarte bleibt immer bei diesem Spieler.
  • Als aktiver Spieler durchläuft man nacheinander folgende vier Phasen:
  1. Bohnenkarte von der Hand anbauen

-Darf nur die vorderste (oberste) Karte sein, danach auch noch eine weitere

-Es können mehrere Bohnen der gleichen Farbe auf verschiedenen Felder angebaut werden, allerdings nicht verschiedene Farben auf einem Feld.

-Sind keine freien Felder vorhanden, so müssen zunächst Felder abgeerntet werden

-Sind zu Beginn von Phase 1 keine Karten auf der Hand, wird direkt in Phase 2 übergegangen

 

  1. Bohnenkarten aufdecken und handeln

-es werden 2 Karten vom Nachziehstapel gezogen und daneben gelegt

-die Felder können abgebaut oder gehandelt werden

-beim Handelt darf nur der aktive Spieler mit seinen Mitspielern handeln, es dürfen nur Handkarten gehandelt werden, „Handelkarten“ dürfen danach nicht weitergehandelt werden, die Anzahl der zuhandelnden Karten darf variieren, Karten dürfen auch verschenkt werden

-die so erhaltenen Karten werden quer neben den Feldern abgelegt

  1. Gehandelte und aufgedeckte Bohnenkarten anbauen

-alle Spieler müssen die Bohnenkarten, die quer neben dem Feld liegen nun anbauen

-der aktive Spieler muss nun auch die restlichen Karten neben dem Ziehstapel anbauen

-sind keine freien Felder auf dem Bohnenfeld vorhanden, müssen zunächst Bohnenkarten abgebaut werden

  1. Bohnenkarte nachziehen

-es werden drei neue Karten gezogen und in der gezogenen Reihenfolge hinter den die Handkarten einsortiert.

Bohnenernte:

  • Bohnen können jeder Zeit geerntet werden
  • So viele Karten, wie es Taler gibt, werden auf den Talerstapel umgedreht gelegt, der Rest der Karten wird aufgedeckt daneben gelegt
  • Bohnenschutzregel: Es dürfen keine einzelnen Bohnenkarten geerntet werden, wenn auf einem Feld mehr als eine Karte liegt bzw. diese Karten müssen dann zuerst geerntet werden.

Ende:

  • Wenn Nachziehstapel zum 3. Mal leer wird.
  • Passiert dies in Phase 2, so wird Phase 2 und 3 zu Ende gespielt.
  • Der Spieler mit den meisten Talern gewinnt, bei Gleichstand, der der am weitesten links vom Startspieler sitzt.
